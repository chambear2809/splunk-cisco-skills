Renders Helm values and install/upgrade scripts; this is the platform install prerequisite for the Splunk Observability + Splunk Platform integration done by splunk-observability-isovalent-integration.

Read and follow the instructions in skills/cisco-isovalent-platform-setup/SKILL.md to help the user. If more detail is needed, also read skills/cisco-isovalent-platform-setup/reference.md.
