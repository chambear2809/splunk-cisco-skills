Install, render, configure, and validate the Splunk add-ons for Microsoft cloud telemetry: Office 365 (splunk_ta_o365) and Microsoft Cloud Services (Splunk_TA_microsoft-cloudservices). Renders Office 365 Management Activity, Microsoft Graph Entra ID metadata, and Azure audit inputs, an Entra app-registration runbook, the o365 and azure indexes, and ingestion validation.

Read and follow the instructions in skills/splunk-microsoft-cloud-setup/SKILL.md to help the user. If more detail is needed, also read skills/splunk-microsoft-cloud-setup/reference.md.
